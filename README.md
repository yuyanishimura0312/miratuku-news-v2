# miratuku-news-v2

miratuku-news の新版。PESTLE analysis-based news platform with future foresight.

## Overview

Original version: https://yuyanishimura0312.github.io/miratuku-news/app.html

This is a redesigned version with improvements planned.

## Structure

- `app.html` — Main application (single-page app, based on v1)
- `data/` — CLA analysis data, signal network data
- `js/` — Shared JavaScript modules
- `admin/` — Admin interface
- `scripts/` — Python data generation scripts
- `worker/` — Cloudflare Worker routes
